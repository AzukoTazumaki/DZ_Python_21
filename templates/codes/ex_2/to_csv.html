Метод добавления слова

Метод to_csv() принимает в качестве аргумента список words_list из метода translate(). Вначале создается список колонок – три языка. Затем используется модуль Pandas, с помощью которого создается DataFrame, в который входит список слов. Затем созданный DataFrame добавляется в файл "words.csv" (В качестве mode используем 'a', чтобы не перезаписывать файл). В конце возвращается вызов метода read_csv_to_html(). 


def to_csv(self, words_list: list): 
    columns: list = ['Russian', 'English', 'French'] 
    df = pd.DataFrame([words_list], columns=columns) 
    df.to_csv('results/words.csv', mode='a', header=False, index=False) 
    return self.read_csv_to_html()
