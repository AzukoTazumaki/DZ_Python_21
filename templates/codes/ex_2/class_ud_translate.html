Инициализация класса UDTranslate

Инициализируем новый класс, который наследуется от класса Translate – UDTranslate (в данном случае UD означает Update & Delete). Затем вызывается метод super(), чтобы вызвать конструктор из главного класса – это позволит вызывать подкласс, передавая в него тот же самый словарь. 


class UDTranslate(Translate): 
    def __init__(self, word: dict): 
        super().__init__(word)
