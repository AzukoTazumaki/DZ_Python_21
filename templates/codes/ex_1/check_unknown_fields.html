Метод проверки на пустоту

Здесь в качестве аргумента из метода add_player_to_csv() приходит список данных, введенных пользователем. Циклом каждый элемент списка с помощью регулярного выражения проверяется на пустоту. Если условие выполнено, элемент заменяется на строку "Неизвестно" – иначе продолжаем перебор. В конце возвращается список отформатированных элементов. Метод является статическим, так как в его теле нет параметра self. 


@staticmethod 
def check_unknown_fields(player_info: list) -> list: 
    for index, field in enumerate(player_info): 
        if len(re.sub(r'\s+', '', field)) == 0: 
            player_info[index] = 'Неизвестно' 
        continue 
    return player_info
